# Extract
Script Bash for extract files

## Install
```
git clone https://github.com/tofoli/extract.git /opt/extract
chmod +x /opt/extract/extract
ln -s /usr/local/bin/extract /opt/extract/extract
```

## Syntax
```
extract file.zip
extract file.rar
extract file.tar.xz
extract file.7z
extract file.ace
```
